myFlix

Documentation

myFlix is an API that enables users to recieve information about movies, directors, and genres of their choosing. The app will invite users to sign up, update personal information, and create lists of their favorite movies.

Resource Information

Response Body Formats JSON
Request Body Formats JSON

Parameters

q Required

How to use this API:

Top Movies Array

Information about "Top Movies" contains the following information:

  • Title
  • Director
    • Director's Name
    • Director's Bio
  • Genre
    • Genre Description
  • Release Year
  • Description
  • Image Url

Users Array

Information about "Top Movies" contains the following information:

  • ID
  • Name
  • Favorite Movies

CREATE

This HTTP request allows the client to create a new user.

POST/PUT

Read HTTP requests allow the client to:

  • Return an list of ALL movies
  • Return movies by {Title}
  • Return descriptions of genres by {Genre Name}
  • Return list of directors by {Director Name}

UPDATE

Put HTTP requests allow the client to update user by ID.

Post HTTP requests allow the user to add {Favorite Movie} title to user.

DELETE

Delete HTTP requests allow the client to:

  • Remove movies from the {Top Movies} array
  • Allow existing users to delete a user from the user array
